## 시작하기

### Ticketing Pro란?

Ticketing Pro는 콘서트, 스포츠, 뮤지컬, 전시회 등 다양한 이벤트의 티켓을 온라인으로 예매할 수 있는 통합 플랫폼입니다.

### 주요 기능

✅ **실시간 좌석 선택**: 좌석 상태를 실시간으로 확인하고 원하는 좌석을 선택
✅ **Virtual Waiting Room**: 인기 공연의 공정한 티켓 구매를 위한 대기열 시스템
✅ **SafeTix**: 60초마다 갱신되는 동적 QR 코드로 위조 방지
✅ **다양한 결제 수단**: 신용카드, 체크카드, Apple Pay, Google Pay 지원
✅ **모바일 월렛 연동**: Apple Wallet, Google Pay에 티켓 저장

### 시스템 요구사항

**웹 브라우저**
- Chrome 90 이상
- Firefox 88 이상
- Safari 14 이상
- Edge 90 이상

**모바일**
- iOS 14 이상 (Safari)
- Android 8.0 이상 (Chrome)

## 회원가입 및 로그인

### 1. 회원가입

1. 홈페이지 우측 상단의 **로그인** 버튼 클릭
2. 로그인 페이지에서 **회원가입** 링크 클릭
3. 다음 정보 입력:
   - **이메일**: 로그인 ID로 사용됩니다
   - **비밀번호**: 최소 8자 이상, 영문/숫자/특수문자 조합
   - **이름**: 실명 (티켓 수령 시 본인 확인용)
   - **전화번호**: SMS 알림 수신용 (010-1234-5678 형식)

4. **회원가입** 버튼 클릭
5. 입력한 이메일로 인증 메일 발송
6. 이메일의 **인증 링크** 클릭하여 계정 활성화

### 2. 로그인

1. 홈페이지 우측 상단의 **로그인** 버튼 클릭
2. 이메일과 비밀번호 입력
3. **로그인** 버튼 클릭

### 3. 비밀번호 찾기

1. 로그인 페이지에서 **비밀번호를 잊으셨나요?** 클릭
2. 가입한 이메일 주소 입력
3. 이메일로 전송된 비밀번호 재설정 링크 클릭
4. 새 비밀번호 입력

## 이벤트 검색하기

### 1. 홈페이지에서 검색

**방법 1: 검색창 사용**
1. 홈페이지 상단 검색창에 아티스트명, 공연명, 장소 등 입력
   - 예: "BTS", "잠실 올림픽", "뮤지컬"
2. **검색** 버튼 클릭 또는 Enter 키 입력

**방법 2: 카테고리 필터**
1. 홈페이지에서 원하는 카테고리 클릭
   - 🎵 **콘서트**: K-POP, 팝, 록, 재즈 등
   - ⚽ **스포츠**: 야구, 축구, 농구 등
   - 🎭 **뮤지컬**: 뮤지컬, 연극 등
   - 🖼️ **전시회**: 미술 전시, 박람회 등

### 2. 검색 결과 필터링

검색 결과 페이지에서 다양한 조건으로 필터링 가능:

- **카테고리**: 공연 종류
- **날짜**: 원하는 날짜 범위
- **가격**: 최소/최대 가격 설정
- **지역**: 서울, 부산, 대구 등
- **상태**: 예매 가능한 공연만 보기

### 3. 정렬 옵션

- **인기순**: 예매율 높은 순
- **최신순**: 등록일 기준
- **가격 낮은 순**
- **가격 높은 순**
- **공연일 빠른 순**

## 티켓 예매하기

### 1. 이벤트 상세 페이지

1. 검색 결과에서 원하는 이벤트 클릭
2. 이벤트 상세 정보 확인:
   - 📅 **일시**: 공연 날짜 및 시간
   - 📍 **장소**: 공연장 위치 및 교통 정보
   - 💰 **가격**: 좌석 등급별 가격
   - ℹ️ **상세 정보**: 공연 소개, 출연진, 주의사항

### 2. 좌석 선택

1. **예매하기** 버튼 클릭
2. 좌석 선택 화면에서 원하는 좌석 클릭
   - 🟢 **초록색**: 예매 가능
   - 🟡 **노란색**: 다른 사용자가 예약 중 (10분 이내)
   - 🔴 **빨간색**: 판매 완료
   - 🔵 **파란색**: 내가 선택한 좌석

3. **좌석 선택 규칙**:
   - 최대 4석까지 선택 가능
   - 같은 섹션 내에서만 선택 가능 (권장)
   - 선택 후 10분 이내 결제 필요

4. **좌석 확인** 버튼 클릭

### 3. 예약 정보 확인

- 선택한 좌석 정보
- 좌석별 가격
- 총 결제 금액
- **남은 시간**: 10분 카운트다운 타이머

## Virtual Waiting Room (대기열)

인기 공연의 경우 공정한 티켓 판매를 위해 대기열 시스템이 적용됩니다.

### 대기열 진입

1. 이벤트 페이지 접속 시 자동으로 대기열 진입
2. 대기열 화면에서 확인 가능한 정보:
   - **현재 내 순번**: 예) 1,523번
   - **예상 대기 시간**: 예) 약 15분
   - **전체 대기 인원**: 예) 5,000명

### 대기 중 주의사항

⚠️ **페이지를 새로고침하지 마세요**
- 새로고침 시 순번이 뒤로 밀립니다

⚠️ **브라우저를 닫지 마세요**
- 창을 닫으면 대기열에서 제외됩니다

✅ **다른 탭에서 작업 가능**
- 대기열 페이지만 열어두면 됩니다

### 입장 가능 알림

1. 순번이 되면 화면이 자동으로 전환됩니다
2. **5분 이내** 좌석 선택 페이지로 이동해야 합니다
3. 시간 초과 시 다시 대기열로 이동

## 결제하기

### 1. 결제 정보 입력

**결제자 정보**
- 이름 (티켓 수령자)
- 이메일 (티켓 전송용)
- 전화번호

**결제 수단 선택**
- 💳 **신용카드/체크카드**
- 🍎 **Apple Pay**
- 📱 **Google Pay**

### 2. 카드 정보 입력 (Stripe)

1. 카드 번호 (16자리)
2. 유효기간 (MM/YY)
3. CVC (3자리)
4. 카드 소유자 이름

### 3. 결제 확인

1. **결제하기** 버튼 클릭
2. 3D Secure 인증 (필요 시)
3. 결제 처리 중... (최대 30초)
4. 결제 완료!

### 4. 결제 완료 후

- ✅ 예매 완료 화면 표시
- 📧 예매 확인 이메일 발송
- 📱 SMS 알림 발송
- 🎫 마이페이지에서 티켓 확인 가능

### 결제 실패 시

**일반적인 원인:**
- 카드 한도 초과
- 잔액 부족
- 잘못된 카드 정보
- 3D Secure 인증 실패

**해결 방법:**
1. 카드 정보 재확인
2. 다른 카드로 시도
3. 카드사에 문의하여 한도 확인

## SafeTix 사용하기

SafeTix는 위조 방지를 위한 동적 QR 코드 시스템입니다.

### 특징

🔄 **60초마다 갱신**
- QR 코드가 자동으로 변경되어 캡처/복사 불가

🔒 **안전한 입장**
- 본인만 사용 가능한 고유 QR 코드

📱 **모바일 최적화**
- 스마트폰 화면으로 간편하게 입장

### 사용 방법

**공연 당일:**

1. **마이페이지 > 내 티켓** 접속
2. 해당 공연 티켓 클릭
3. QR 코드 화면 표시
4. 공연장 입구에서 QR 리더기에 인식
5. ✅ 입장 완료!

**주의사항:**

⚠️ **스크린샷 불가**
- QR 코드는 실시간으로 변경되므로 캡처 무의미

⚠️ **화면 밝기 조절**
- QR 인식을 위해 화면 밝기를 최대로 설정

⚠️ **배터리 확인**
- 공연 전 스마트폰 충전 상태 확인

### 모바일 월렛 추가

**Apple Wallet (iOS)**
1. 티켓 상세 페이지에서 **Apple Wallet에 추가** 클릭
2. Wallet 앱에서 티켓 확인
3. 잠금 화면에서 빠르게 접근 가능

**Google Pay (Android)**
1. 티켓 상세 페이지에서 **Google Pay에 추가** 클릭
2. Google Pay 앱에서 티켓 확인

## 취소 및 환불

### 취소 정책

**전액 환불 (100%)**
- 공연 7일 전까지

**부분 환불 (90%)**
- 공연 3~6일 전 (취소 수수료 10%)

**부분 환불 (80%)**
- 공연 1~2일 전 (취소 수수료 20%)

**환불 불가**
- 공연 당일 및 공연 종료 후

### 취소 방법

1. **마이페이지 > 내 티켓** 접속
2. 취소할 티켓 선택
3. **예매 취소** 버튼 클릭
4. 취소 사유 선택
5. **확인** 버튼 클릭

### 환불 처리

- **카드 결제**: 3~5 영업일 내 카드 승인 취소
- **계좌 이체**: 신청 시 입력한 계좌로 입금 (2~3 영업일)

### 주최사 사정으로 인한 취소

공연 취소 시:
- ✅ 전액 환불 (수수료 없음)
- 📧 이메일 및 SMS 알림 발송
- 🔄 자동 환불 처리 (7영업일 이내)

## FAQ

### Q1. 회원가입 없이 예매 가능한가요?
**A.** 아니요. 티켓 구매를 위해서는 회원가입이 필수입니다. 본인 인증 및 예매 내역 관리를 위해 필요합니다.

### Q2. 예매 후 좌석 변경이 가능한가요?
**A.** 예매 완료 후에는 좌석 변경이 불가능합니다. 변경을 원하시면 기존 예매를 취소하고 다시 예매해야 합니다.

### Q3. 한 번에 몇 장까지 예매할 수 있나요?
**A.** 최대 4장까지 예매 가능합니다. 단, 공연에 따라 1인당 구매 제한이 있을 수 있습니다.

### Q4. 예매 후 10분 안에 결제하지 못하면 어떻게 되나요?
**A.** 예약이 자동 취소되고 선택한 좌석은 다시 판매됩니다. 처음부터 다시 예매해야 합니다.

### Q5. 티켓 양도가 가능한가요?
**A.** 일부 공연은 양도 기능을 제공합니다. 마이페이지에서 **티켓 양도** 메뉴를 확인하세요.

### Q6. QR 코드를 캡처해서 사용할 수 있나요?
**A.** 아니요. SafeTix는 60초마다 QR 코드가 변경되므로 캡처 화면은 사용할 수 없습니다.

### Q7. 스마트폰이 없으면 입장이 불가능한가요?
**A.** 마이페이지에서 티켓을 인쇄할 수 있습니다. 인쇄된 티켓의 QR 코드로도 입장 가능합니다.

### Q8. 외국인도 예매할 수 있나요?
**A.** 네, 가능합니다. 단, 본인 확인을 위해 여권이 필요할 수 있습니다.

### Q9. 결제 후 영수증을 받을 수 있나요?
**A.** 네. 마이페이지 > 예매 내역에서 **영수증 출력** 버튼을 클릭하시면 됩니다.

### Q10. 미성년자도 예매할 수 있나요?
**A.** 14세 이상은 가능하며, 14세 미만은 법정대리인 동의가 필요합니다.
