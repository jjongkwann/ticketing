## 2. 주요 컴포넌트 명세

### 2.1 Button (버튼)

**Variants:**
- **Primary**: 주요 액션 (예매하기, 결제하기)
- **Secondary**: 보조 액션 (취소, 이전)
- **Outline**: 테두리만 있는 버튼
- **Ghost**: 배경 없는 버튼
- **Link**: 링크 스타일 버튼

**Sizes:**
- **Small**: 32px 높이, 12px ~ 16px 패딩
- **Medium**: 40px 높이, 16px ~ 20px 패딩 (기본)
- **Large**: 48px 높이, 20px ~ 24px 패딩

**States:**
- Default
- Hover (색상 darkening 또는 shadow)
- Active (약간 scale down)
- Disabled (투명도 50%)
- Loading (스피너 표시)

---

### 2.2 Card (카드)

**이벤트 카드:**
```
┌─────────────────┐
│   [포스터 이미지] │  ← 16:9 비율
├─────────────────┤
│ 이벤트명         │  ← 1줄, 초과 시 ellipsis
│ 📅 2024.05.20   │  ← 아이콘 + 날짜
│ 📍 잠실 주경기장  │  ← 아이콘 + 장소
│ ₩50,000 ~       │  ← 가격
│ 🎫 500석 남음    │  ← 잔여 좌석
└─────────────────┘
```

**예약 카드:**
```
┌────────────────────────────┐
│ [썸네일] BTS World Tour     │
│          📅 2024.05.20      │
│          🪑 VIP석 A3, A4    │
│          상태: ✅ 예약확정   │
│          [티켓 보기] [취소]  │
└────────────────────────────┘
```

---

### 2.3 Input (입력 필드)

**Text Input:**
```
┌─────────────────────────────┐
│ 라벨 (선택적)                │
├─────────────────────────────┤
│ [입력 텍스트...]            │
│  ↑ Placeholder              │
└─────────────────────────────┘
  에러 메시지 또는 도움말
```

**States:**
- Default: Border gray
- Focus: Border primary color, 외곽선 표시
- Error: Border red, 에러 메시지 표시
- Disabled: 배경 gray, 입력 불가

**Types:**
- Text
- Email
- Password (눈 아이콘으로 보이기/숨기기)
- Number
- Tel
- Date

---

### 2.4 Modal (모달)

**구조:**
```

├─────────────────────────────────────────┤
│                                         │
│  내용 영역                               │
```

**사용 예:**

- 예약 취소 확인
- 에러 메시지
- 알림

---

### 2.5 Timer (타이머)

**표시 형식:**
```
⏰ 09:45
```

**색상 변화:**
- 10분 ~ 3분: 기본 색상
- 3분 ~ 1분: 주황색 (warning)
- 1분 ~ 0분: 빨간색 (danger) + 깜빡임

**인터랙티브 요소:**

- 확대/축소 버튼
- 드래그로 이동 (Pan)
- 구역별 필터

**좌석 상태 색상:**
- 🟢 Green (#10B981): 예약 가능
- 🟡 Yellow (#F59E0B): 임시 예약 (다른 사용자)
- 🔴 Red (#EF4444): 예약 완료
- 🔵 Blue (#3B82F6): 현재 사용자 선택
